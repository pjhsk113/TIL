# [Java8] 소개 및 학습 목차

# Java8 소개

Java8은 LTS로써 2014년 3월에 등장했다. 현재 자바 개발자 중 83%가 사용중일 정도로 많이 사용되고 있는 버전이다. [JetBrains 통계 내용](https://www.jetbrains.com/lp/devecosystem-2019/java/)

# Java8에 추가된 주요 기능

- 람다 표현식
- 메소드 레퍼런스
- Stream API
- Optional<T>
- default 메서드, static 메서드
- Date와 Time API
